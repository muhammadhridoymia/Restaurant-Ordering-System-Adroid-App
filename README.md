# Restaurant Ordering System – Android App

A mobile application designed to simplify the restaurant ordering process by allowing waiters or customers to place orders digitally and send them directly to the kitchen.

In many restaurants, when customers arrive, waiters take orders manually by writing on paper and then communicating the order to the kitchen. During busy hours, this process becomes slow, difficult, and error-prone, creating pressure for waiters and confusion in the kitchen.

This application solves that problem by providing a digital ordering system.

Restaurants can place a **phone or tablet on each table**, allowing customers to browse the menu and place their own orders. It's has one Web App too, so customers can order or visit by QR code .

The system helps to:

- Reduce ordering mistakes  
- Speed up communication between tables,waiters and kitchen
- Improve restaurant efficiency during busy hours  
- Provide a better customer experience
